# === BACKEND (API) ===

# Para rodar o servidor da API localmente:

# 1. Abra um terminal na pasta raiz do projeto (compartilha-ai).

# 2. Ative o ambiente virtual (venv). No Windows:
.\venv\Scripts\activate

# 3. Inicie o servidor com Uvicorn:
uvicorn backend.main:app --reload --host 0.0.0.0 --port 8001

# O que cada parte faz:
# --reload: O servidor reinicia automaticamente quando você altera o código.
# --host 0.0.0.0: Permite que o servidor seja acessado de outros dispositivos na sua rede (como seu celular).
# --port 8001: Define a porta em que o servidor vai rodar.

# ---
# Para desativar o ambiente virtual depois que terminar:
deactivate


# === FRONTEND (React App) ===

# Para rodar a aplicação frontend localmente:

# 1. Abra um segundo terminal e navegue até a pasta do frontend:
cd frontend

# 2. Instale as dependências (só precisa fazer isso uma vez):
npm install

# 3. Inicie o servidor de desenvolvimento:
npm run dev
